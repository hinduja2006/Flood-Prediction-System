# Flood Prediction System

## Project Description

This project predicts the possibility of floods using Machine Learning based on weather and rainfall parameters.

## Technologies Used

- Python
- Flask
- Scikit-learn
- XGBoost
- HTML
- CSS
- JavaScript

## Project Structure

app.py

templates/

static/

floods.save

transform.save

requirements.txt

## Installation

1. Install Python

2. Install dependencies


pip install -r requirements.txt


3. Run the application


python app.py


4. Open


http://127.0.0.1:5000
